Aegis Code Forge

NOTE: This project has no relations with the original Aegis.

Aegis is a self-hosted code forge that mainly supports the Git distributed version control system.

Latest Release: v0.1.3 (2025-09-16)
Latest Nightly: v0.1.3-d52b96ab (2025-09-16)

  • Simple, functional UI;
  • Support read-only clone by HTTP and read/write access through SSH;
  • Support for issue tracking and pull requests;
  • Support for namespaces;
  • Support for per-namespace and per-repository user access control;
  • No JavaScript - works all major browsers and a lot of minor ones, including Ladybird and NetSurf.
  • Proper web installer for easy(-er) installation;
